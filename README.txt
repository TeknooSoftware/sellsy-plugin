Form generator to acquire lead and save theme into Sellsy account

== Description ==
The Wordpress plugin Sellsy allow you to display a contact form connected to your Sellsy account. When the form will be submited, a prospect and optionally an opportunity will be created on your Sellsy account.

== Installation ==
Copy this plugin into plugin directory. Enable this plugin and go to the admin panel of this plugin, called \"Sellsy\".
Then follow displayed instructions to set your sellsy api keys.
